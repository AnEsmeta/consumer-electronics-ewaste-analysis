# Visual Data Analysis of Consumer Electronics Purchasing Behaviour, E-Waste Trends and Interest in Refurbished Devices

## Author
Aneeta Parayanthra Sebastian

## Project Overview

This project investigates consumer electronics purchasing behaviour, global e-waste trends, and interest in refurbished electronics. The goal is to identify relationships between consumer demand, sustainability, and electronic waste generation using visual data analysis.

## Research Questions

1. Are consumers showing increasing interest in refurbished electronics over time?
2. Which customer factors influence online purchasing behaviour?
3. How does electronics demand relate to e-waste growth?
4. Which factors influence the choice between new and refurbished products?
5. How can consumer behaviour data support sustainable market strategies?
